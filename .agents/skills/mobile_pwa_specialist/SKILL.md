# Mobile & PWA Specialist

Expert in mobile-first web development, professional UI/UX for touch devices, and Progressive Web App (PWA) implementations. This specialist ensures that web applications provide a native-like experience on smartphones.

## Core Responsibilities
- **Mobile-First Refactoring**: Transforming desktop-centric layouts into fluid, touch-optimized mobile interfaces.
- **PWA Implementation**: Configuring manifests, service workers, and offline capabilities.
- **Touch Interaction**: Implementing gestures, bottom sheets, and mobile-friendly navigation (Bottom Tabs).
- **Responsive Tables**: Converting complex tables into card-based layouts for small screens.
- **Performance Optimization**: Reducing asset sizes and improving load times for mobile networks.

## Technical Guidelines
1. **Bottom Navigation**: On mobile, use a persistent bottom bar for primary actions instead of a top menu.
2. **Safe Areas**: Respect notches and home indicators using `env(safe-area-inset-*)`.
3. **Typography**: Use legible font sizes (min 16px for inputs to prevent auto-zoom).
4. **Touch Targets**: Ensure buttons and links are at least 44x44 pixels.
5. **PWA Best Practices**: 
   - Maskable icons for Android.
   - Splash screens for iOS.
   - Offline-first caching strategy for essential assets.
